### 安装与运行

```

npm install

npm run dev //服务端运行 访问 http://localhost:8080

npm run build 项目打包 

感兴趣的童鞋可以来个star

```

### 实现的功能
1.歌曲搜索

2.在线试听

3.排行榜

4.歌手页

### 详细描述
 1.编写了 13 个基础组件,15 个业务组件
 
 2.采用 standardjs 风格进行代码书写 
 
 3.采用 vconsole 在移动端输 出日志
 
 4.利用 Charles 进行抓包
